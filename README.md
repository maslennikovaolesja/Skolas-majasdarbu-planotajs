Skolas mājasdarbu plānotājs

Projekta apraksts

"Skolas mājasdarbu plānotājs" ir tīmekļa lietotne, kas paredzēta skolēniem mājasdarbu plānošanai un pārvaldībai.
Lietotne palīdz sekot līdzi uzdotajiem mājasdarbiem, to izpildes termiņiem un izpildes statusam.

Projekta mērķis

Projekta mērķis ir izveidot vienkāršu un ērti lietojamu sistēmu, kas palīdz skolēniem organizēt savus mājasdarbus un samazināt iespēju aizmirst par uzdotajiem darbiem.

Galvenās funkcijas:
1. Mājasdarbu pievienošana
2. Mājasdarbu apskate
3. Mājasdarbu dzēšana
4. Mājasdarbu atzīmēšana kā izpildītiem
5. Filtrēšana pēc priekšmeta
6. Šodienas uzdevumu apskate
7. Datu saglabāšana pārlūkprogrammā

Izmantotās tehnoloģijas:
1. HTML5
2. CSS3
3. JavaScript
4. LocalStorage

Projekta palaišana:
1. Lejupielādējiet projekta failus
2. Atveriet failu index.html interneta pārlūkprogrammā
3. Sāciet lietot sistēmu
